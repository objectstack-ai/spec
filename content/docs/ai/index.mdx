# AI & Automation

ObjectStack provides comprehensive AI and automation capabilities to accelerate development and enable intelligent applications.

## ObjectStack Builder

The **ObjectStack Builder** is an AI-driven system that automates application development through metadata generation.

### Key Features

- **Natural Language to Code**: Describe requirements in plain language, get valid metadata
- **Automated Git Workflow**: Automatic branch creation, commits, and pull requests
- **Self-Healing Deployments**: Detect and fix deployment errors automatically
- **Protocol Compliance**: Strict adherence to P0 specifications

### Learn More

- [Builder Workflow](./builder-workflow) - Development lifecycle and automation
- [Technical Specification](./technical-specification) - Detailed architecture and implementation

## AI Protocol

ObjectStack includes native AI capabilities for building intelligent applications:

- **Agent System**: Define autonomous agents with tools and knowledge bases
- **RAG Pipeline**: Retrieval-augmented generation for context-aware AI
- **Model Registry**: Manage and route between multiple LLM providers
- **Conversation Management**: Track and manage multi-turn conversations

See the [AI Protocol Reference](../references/ai) for complete specifications.

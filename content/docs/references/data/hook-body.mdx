Capability tokens a script body may request.

The runtime sandbox enforces these ‚Äî if a body uses a `ctx` API that requires

a capability it did not declare, the call throws at invocation time.

- `api.read`   ‚Äî `ctx.api.object(...).find / findOne / count / aggregate`

- `api.write`  ‚Äî `ctx.api.object(...).insert / update / delete`

- `crypto.uuid` ‚Äî `ctx.crypto.randomUUID()`

- `crypto.hash` ‚Äî `ctx.crypto.hash(algo, data)`

- `log`        ‚Äî `ctx.log.info / warn / error`

`http.fetch` is intentionally absent ‚Äî outbound calls go through Connector

recipes (separate spec) so they remain auditable and replayable.

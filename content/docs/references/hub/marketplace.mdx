# Marketplace Protocol

Defines the schema for the ObjectStack Plugin Marketplace.

This is the catalog of available extensions sourced from NPM or private registries.

<Callout type="info">
**Source:** `packages/spec/src/hub/marketplace.zod.ts`
</Callout>

## TypeScript Usage

```typescript
import { MarketplacePlugin, PluginAuthor, PluginPricing, PluginVersion } from '@objectstack/spec/hub';
import type { MarketplacePlugin, PluginAuthor, PluginPricing, PluginVersion } from '@objectstack/spec/hub';

// Validate data
const result = MarketplacePlugin.parse(data);
```

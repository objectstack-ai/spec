## MultipartUploadConfig

### Properties

| Property | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| **enabled** | `boolean` | ✅ | Enable multipart uploads |
| **partSize** | `number` | ✅ | Part size in bytes (min 5MB) |
| **maxConcurrentParts** | `number` | ✅ | Maximum concurrent part uploads |
| **threshold** | `number` | ✅ | File size threshold for multipart upload in bytes |

## Common Patterns

### Naming Conventions

<Tabs items={['Configuration Keys', 'Data Values', 'Schema Names']}>
  <Tab value="Configuration Keys">
    ```typescript
    // Property names in schemas: camelCase
    {
      maxLength: 100,
      defaultValue: 'example',
      referenceFilters: ['active = true']
    }
    ```
  </Tab>
  <Tab value="Data Values">
    ```typescript
    // Machine identifiers in data: snake_case
    {
      name: 'project_task',
      object: 'customer',
      field: 'first_name'
    }
    ```
  </Tab>
  <Tab value="Schema Names">
    ```typescript
    // Export names: PascalCase + Schema suffix
    export const FieldSchema = z.object({...});
    export type Field = z.infer<typeof FieldSchema>;
    ```
  </Tab>
</Tabs>

### Common Imports

```typescript
// Data protocols
import { FieldSchema, ObjectSchema, QuerySchema } from '@objectstack/spec/data';

// UI protocols
import { ViewSchema, AppSchema, DashboardSchema } from '@objectstack/spec/ui';

// System protocols

```

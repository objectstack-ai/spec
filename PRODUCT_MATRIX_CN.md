## 🚀 快速开始

### 开发者入门

```bash
# 1. 克隆仓库
git clone https://github.com/objectstack-ai/spec.git
cd spec

# 2. 安装依赖
pnpm install

# 3. 构建所有包
pnpm build

# 4. 运行测试
pnpm test

# 5. 启动 Studio
pnpm studio
```

### 应用开发

```bash
# 创建新项目
npx @objectstack/cli init my-app
cd my-app

# 启动开发服务器
os dev

# 启动可视化控制台
os studio
```

# @objectstack/objectui

UI Layer for ObjectStack - View rendering and component system.

## Overview

This package provides the UI implementation layer for ObjectStack, including:

- View rendering (List, Form, Dashboard, Report)
- Component system
- Layout engine
- Theme system

## Installation

```bash
pnpm add @objectstack/objectui
```

## Usage

```typescript
import { version } from '@objectstack/objectui';

console.log('ObjectUI version:', version);
```

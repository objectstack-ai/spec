# Changelog

## [1.0.0] - 2026-01-21

### Added
- Initial release of Better-Auth authentication plugin
- Support for multiple authentication strategies (email/password, OAuth, magic link, passkey, OTP, anonymous)
- 10+ OAuth providers (Google, GitHub, Facebook, Twitter, LinkedIn, Microsoft, Apple, Discord, GitLab, custom)
- Advanced security features (rate limiting, CSRF protection, 2FA, session fingerprinting)
- Database adapter support (Prisma, Drizzle, Kysely, custom)
- Email provider integration (SMTP, SendGrid, Mailgun, AWS SES, Resend, custom)
- Lifecycle hooks for authentication events
- Comprehensive examples and documentation
